Progressão aritimética

Definição:É uma sequência numérica, onde um de seus termos, a partir do segundo é igual à soma do anterior com uma constante r, denomindando a razão da PA.

  • r = Razão que existe dentro da PA
  • a2 = Segundo Termo da PA
  • a1 = Primeiro Termo da PA
  • an = enésimo termo
  • n = número de termos

Fórmulas da PA

  • Razão da PA -> r=a2-al
  • termos -> PA= (al+r), (a1+2)_[a1 + (n-10 r]
  • Soma dos termos -> Sn=(a1+a2)*n/2
  • termo médio ->Am = a1 + an /2

Tipos de PA

Constante: razão zero e os termos são iguais PA=(2,2,2,2,2,... ) onde r=0

Crescente: razão maior que zero e termo maior que o anterior PA=(2,4,5,8,10), onde r=2

Decrescente: razão menos que zero e o termo menor que o anterior PA=(4,2,0,-2,-4) onde r=-2

Exemplos

Qual é o 50º termo de uma PA(3,5,7,...)?

An= a1 + (n-1)r

A50=3+(50-1)2

A50=3+49*2

A50= 3+ 98

A50= 101

Análise: Neste exercício deveríamos descobrir o 50º número, como ja tínhamos os valores que seriam utilizados na fórmula, apenas os substituímos: a1=3, n=50, r=2. Podemos concluir que o r corresponde ao 2 porque a distância entre 3 e 5 é de 2, para chegar a esse resultado poderíamos subtrair 5-3=2.

Interpolação Geométrica

A interpolação aritmética é um método para encontrar um termo intermediário em uma progressão aritmética (PA). A PA é uma sequência numérica em que cada termo é igual ao termo anterior somado a uma constante chamada de razão. Para interpolar um termo intermediário em uma PA, precisamos de três informações:

  • O primeiro termo da PA (a1)
  • O último termo da PA (an)
  • A razão da PA (r)

Com essas informações, podemos calcular o termo intermediário usando a seguinte fórmula:

t = a1 + (n-1)*r

Onde:

t é o termo intermediário

n é o número do termo intermediário

Soma dos termos de uma PA

A soma dos termos de uma progressão aritmética (PA) é igual a:

Sn = n/2 * (a1 + an)

Onde:

  • Sn é a soma dos termos
  • n é o número de termos
  • a1 é o primeiro termo
  • an é o último termo

Diferenças entre Interpolação Aritmética e Geométrica

A principal diferença entre a interpolação aritmética e a interpolação geométrica é que a primeira é usada para progressões aritméticas (PA), enquanto a segunda é usada para progressões geométricas (PG). Em uma PA, cada termo é igual ao termo anterior somado a uma constante chamada de razão. Em uma PG, cada termo é igual ao termo anterior multiplicado por uma constante chamada de razão. A interpolação aritmética usa a seguinte fórmula para encontrar um termo intermediário:

t = a1 + (n-1)*r

    Onde

  • t é o termo intermediário
  • n é o número do termo intermediário
  • a1 é o primeiro termo
  • r é a razão

A interpolação geométrica usa a seguinte fórmula para encontrar um termo intermediário:

t = a * r^(n-1)

    Onde

  • t é o termo intermediário
  • n é o número do termo intermediário
  • a é o primeiro termo
  • r é a razão
